#### 综述

以上问题的共性都是无法用数学的公式或方程来描述，是一些“非数值计算“的程序设计问题。

描述非数值计算问题的数学模型不是数学方程，而是例如表、树、图这些有逻辑关系的数据。

数据结构是一门研究非数值计算的程序设计中计算机的操作对象以及它们之间的关系和操作的学科
